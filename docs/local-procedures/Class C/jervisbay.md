## Airspace
YJBY is an **uncontrolled aerodrome** which sits inside the temporarily activated Nowra [restricted area](../../airspace/sua.md#restricted-areas). When **NW TCU** is online, various airspace activations reclassify the Class G airspace above YJBY to Class C, including R421A which extends to the surface.

### TCU Closed Procedures
When NW TCU is offline, the restricted airspace above YJBY from `SFC` to `A085` is reclassified as Class G, and from `A085` to `F180` reclassified as Class E. YJBY reverts to CTAF procedures.

## CTA Operations
### Departures
Outbound aircraft shall report when taxiing for departure to NW TCU. Controllers will issue the aircraft with a squawk code and a traffic statement regarding any aircraft operating on, or shortly to land at, the aerodrome.

!!! note
    Both VFR and IFR aircraft require a clearance to operate in Class C airspace. The examples below show an IFR aircraft departing.

!!! phraseology
    **AM301**: "Nowra Approach, AM301, King Air, IFR, taxiing YJBY for YMEN, runway 26"  
    **ESA**: "AM301, squawk 4432, no reported traffic, report lined up for airways clearance"  
    **AM301**: "Squawk 4432, wilco, AM301"

In response to your ready call, NW TCU will issue an airways clearance, if available.

!!! phraseology
    **AM301**: "AM301, lined up"  
    **ESA**: "AM301, cleared to YMEN via URBOB, flight plan route, climb to `A100`"  
    **AM301**: "Cleared to YMEN via URBOB, flight plan route, climb to `A100`, AM301"

All surface areas at YJBY are outside TCU's jurisdiction. Takeoff clearances are not given.

Once airborne, make a Class C [airborne report](../../radio-telephony/Controlled%20Airspace/terminal.md#first-contact) to NW TCU.

### Arrivals
Inbound aircraft shall request airways clearance from NW TCU (unless already in ENR CTA). Aircraft may be cleared for a visual approach or one of YSNW's instrument approaches (expecting to then track to YJBY when visual).

As a landing clearance is not given, aircraft will instead be informed of any traffic operating on the aerodrome and instructed to *'report clear of the runway'*, when their SARWATCH (for IFR aircraft) will be terminated.

!!! phraseology
    **AM301**: "AM301, visual"  
    **ESA**: "AM301, cleared visual approach, no reported IFR traffic, report clear of the runway"  
    **AM301**: "Cleared visual approach, AM301"  
    ...  
    **AM301**: "AM301, clear of the runway, YJBY, cancel SARWATCH"  
    **ESA**: "AM301, YJBY SARWATCH terminated"
