## Airspace
YWSL is an **uncontrolled aerodrome** which sits inside the temporarily activated East Sale [restricted area](../../airspace/sua.md#restricted-areas). When **ES TCU** is online, various airspace activations reclassify the Class G airspace above YWSL to Class C, including R360A which extends to the surface.

### TCU Closed Procedures
When ES TCU is offline, the restricted airspace above YWSL from `SFC` to `F125` is reclassified as Class G, and from `F125` to `F210` (and above) reclassified as Class E. YWSL reverts to CTAF procedures.

## CTA Operations
### Departures
Outbound aircraft shall report when taxiing for departure to ES TCU. Controllers will issue the aircraft with a squawk code and a traffic statement regarding any aircraft operating on, or shortly to land at, the aerodrome.

!!! note
    Both VFR and IFR aircraft require a clearance to operate in Class C airspace. The examples below show an IFR aircraft departing.

!!! phraseology
    **AM301**: "Sale Approach, AM301, King Air, IFR, taxiing YWSL for YMEN, runway 27"  
    **ESA**: "AM301, squawk 4432, no reported traffic, report lined up for airways clearance"  
    **AM301**: "Squawk 4432, wilco, AM301"

In response to your ready call, ES TCU will issue an airways clearance, if available.

!!! tip
    A radar SID is available at YWSL for use when ES TCU is online. Expect to be cleared via the **Radar SID** or a **visual departure**.

!!! phraseology
    **AM301**: "AM301, lined up"  
    **ESA**: "AM301, cleared to YMEN via DUNNE, flight plan route, WS2 departure, climb via SID to `F150`"  
    **AM301**: "Cleared to YMEN via DUNNE, flight plan route, climb to `F150`, AM301"  
    **ESA**: "AM301, assigned heading left 360, report airborne"  
    **AM301**: "Assigned heading left 360, AM301"  

All surface areas at YWSL are outside TCU's jurisdiction. Takeoff clearances are not given.

Once airborne, make a Class C [airborne report](../../radio-telephony/Controlled%20Airspace/terminal.md#first-contact) to ES TCU.

### Arrivals
Inbound aircraft shall request airways clearance from ES TCU (unless already in ENR CTA). Aircraft may be cleared for a visual approach or one of West Sale's instrument approaches.

As a landing clearance is not given, aircraft will instead be informed of any traffic operating on the aerodrome and instructed to *'report clear of the runway'*, when their SARWATCH (for IFR aircraft) will be terminated.

!!! phraseology
    **AM301**: "AM301, visual"  
    **ESA**: "AM301, cleared visual approach, no reported IFR traffic, report clear of the runway"  
    **AM301**: "Cleared visual approach, AM301"  
    ...  
    **AM301**: "AM301, clear of the runway, YWSL, cancel SARWATCH"  
    **ESA**: "AM301, YWSL SARWATCH terminated"
