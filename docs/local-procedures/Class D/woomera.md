## Airspace
**WR ADC** is responsible for the Class D airspace within the **R222F** [restricted area](../../airspace/sua.md#restricted-areas), `SFC` to `F120`.

### Tower Closed Procedures
When **WR ADC** is offline, the WR CTR (Class D `SFC` to `F120`) reverts to Class G. CTAF procedures apply at YPWR.

## Departures
VFR aircraft should expect to depart via a visual departure, on track to their first tracking point.

IFR aircraft shall expect to depart via a [pilot-managed](../../navigation/ifrdepartures.md#other-departure-methods) IFR departure or **visual departure**.

!!! note
    There is 500ft of Class G airspace between **WR ADC** Class D and **Melbourne Centre** Class E. Depending on traffic levels, aircraft departing the CTR on climb may either be instructed to leave and re-enter CTA on climb, or instructed to leave CTA on climb with an expectation of onwards clearance with Melbourne Centre.

    IFR aircraft **must not** enter the overlying Class E airspace until in receipt of an airways clearance.

## Arrivals
RNP approaches are available to both runways. An NDB approach is available to RWY 18.

IFR aircraft can generally expect to be processed direct to the IAF for the following approach:

| Runway | Approach |
| --- | --- |
| RWY 18 | RNP |
| RWY 36 | RNP |

!!! note
    There is 500ft of Class G airspace between **WR ADC** Class D and **Melbourne Centre** Class E. IFR aircraft cruising above `F125` will be instructed to leave CTA on descent by Melbourne Centre and instructed to contact **WR ADC** for onwards clearance.

    Aircraft **must not** enter the Class D CTR until in receipt of an airways clearance from WR ADC.

### Initial and Pitch
The intial points are are aligned with Taxiway C at the following locations.

| RWY  | Initial Point | Altitude |
| ---- | ------------- | ------------------------ |
| 18   | 3NM north, at Yandandarre Creek | `A025` |
| 30   | 2NM south, at Camp Rapier | `A025`      |

## Special Use Airspace
### Military Gates
There are several military gates established throughout the WR CTR to facilitate entry and exit to adjoining SUA.

| Visual Gate         | Position |
| ------------------- | -------- |
| Island Lagoon       | WR180013 |
| Koolymilka Lake     | WR300017 |
| Lake Hart           | WR249022 |
| Lake Richardson     | WR061009 |
| Lake Windabout      | WR121016 |
| Red Swamp           | WR004018 |

!!! note
    Details of each gate can be found in the [Woomera FIHA](https://ais-af.airforce.gov.au/australian-aip) and on the Woomera WAC.
